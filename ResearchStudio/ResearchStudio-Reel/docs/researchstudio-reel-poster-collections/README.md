# ResearchStudio-Reel Poster Collections

100 single-page academic conference posters generated by the ResearchStudio
**paper2poster** pipeline on the Paper2Poster 100-paper evaluation set. Provided
for use as a benchmark / comparison set.

Each paper ships three formats: `poster.png` (preview), `poster.pdf` (print,
true 60x36 in), and `poster.pptx` (native, editable PowerPoint). Math is typeset
with KaTeX; the PPTX is editable text + shapes, not a flat image.

## Contents

```
posters/<slug>/poster.{png,pdf,pptx}   # 100 papers x 3 formats
index.csv / index.json                 # slug <-> title / venue / arXiv / code
README.md
gallery.html                           # offline 3-per-row viewer
```

`index.csv` (also in this folder) maps each poster to its paper.
